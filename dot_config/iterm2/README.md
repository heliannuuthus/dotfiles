# iTerm2 配置说明

## 1. 黑绿极客主题

1. 打开 iTerm2 → **Settings** (⌘,) → **Profiles** → **Colors**
2. 点击 **Color Presets** → **Import...**
3. 选择 `Geek-Black-Green.itermcolors`（本目录下）
4. 在 Presets 下拉中选中 **Geek-Black-Green**

## 2. Option + 左/右箭头 跳词

1. **Settings** → **Profiles** → **Keys**
2. 点击 **Key Mappings** 下方的 **Presets...**
3. 选择 **Natural Text Editing**

该预设会启用：
- **Option + ←**：跳到上一个词
- **Option + →**：跳到下一个词
- **Option + ⌫**：删除上一个词
- **fn + ←/→**：行首/行尾

若 Presets 里没有，需确保 iTerm2 为最新版（`brew upgrade --cask iterm2`）。
